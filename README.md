# 🫁 PulmoAI - AI Lung X-Ray Analyzer

PulmoAI is an AI-powered web application that analyzes lung X-ray images and validates whether the uploaded image is a valid chest X-ray before prediction.

---

## 📌 Features

- 🩻 Lung X-ray validation
- 🤖 AI-based prediction
- 🚫 Invalid image detection
- 🔐 Login & Authentication
- 📊 Dashboard interface
- 📁 Upload history tracking

---

## 🛠️ Technologies Used

- Python
- Flask
- PyTorch
- HTML
- CSS
- JavaScript
- Bootstrap

```bash

```
